# 云南亲子旅行任务手册 🌄

一个为小朋友设计的旅行互动网页应用，让旅行变成一场探险游戏！

## ✨ 功能特点

- 🎮 **像素风格设计** - 复古游戏风格，吸引孩子注意力
- ⭐ **星星收集系统** - 完成任务收集星星，兑换实物奖励
- 📷 **照片上传** - 记录旅行中的精彩瞬间
- ✍️ **日记功能** - 培养写作习惯，留下旅行回忆
- 👨‍👩‍👧 **家长确认机制** - 确保任务质量
- 📊 **进度追踪** - 实时查看完成情况
- 🏆 **奖励系统** - 50/100/200颗星解锁不同奖励

## 📚 旅行行程

11天云南自驾游：
1. 昆明·滇池 - 高原湖泊
2. 梁王山 - 山岳风光
3. 抚仙湖·建水 - 湖泊+古城
4. 碧色寨·蒙自 - 米轨铁路+过桥米线
5. 元阳梯田 - 世界遗产
6. 哈尼小镇·坝达 - 民族文化
7. 多依树·爱春 - 梯田日出
8. 个旧·加级寨 - 工业城市+梨花
9. 石屏·异龙湖 - 湿地+海菜腔
10. 宝秀·郑营 - 古建筑群
11. 昆明·总结 - 旅程回顾
